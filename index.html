MEG Encoder - Contrastive Learning for Neuroimaging

github.com/yiheinchai/meg

Developed self-supervised learning system for magnetoencephalography (MEG) brain signals using contrastive learning via NTXentLoss.

Technical Implementation:

  • Dataset: CamCAN MEG resting-state data (700+ subjects, 306 channels @ 1000Hz).
  • Architecture: 1D CNN encoder (Conv1D → ReLU → AdaptiveAvgPool → FC) projecting 306-channel timeseries to 128D embeddings.
  • NTXent Contrastive Loss: Trained pairs of augmented windows from same subject to have similar embeddings.
  • Data Pipeline: Built HDF5-based caching system eliminating I/O bottlenecks (100x speedup over raw .fif files).
